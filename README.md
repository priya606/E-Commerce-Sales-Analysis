# E-Commerce-Sales-Analysis

###  E-Commerce Sales Analysis Pipeline

####   Overview
An end-to-end data analysis pipeline built using Python and Pandas on 1000+ e-commerce orders simulating real-world retail data.

####  Tech Stack
- Python
- Pandas
- NumPy

#### Analysis Performed
- Data cleaning — null handling, data type fixing, datetime extraction
- Feature engineering — discount-adjusted amounts, order size buckets, return flags
- Business insights — top revenue categories, city-wise return rates, peak hours, top 10 customers by spend
- Anomaly detection — flagged suspicious orders based on amount, quantity, and repeated customer behaviour
- Exported clean dataset and summary report as CSV
